Simon Pilkington

Talk details

Title

Building the next version of the Smoke Framework

Description

Last year the Encoding Platform team within Prime Video launched a number of web services written in Swift using the Smoke Framework. This talk will start by looking back at the development and motivation for the Smoke Framework and discuss our experiences of launching in production. The talk will finish by looking forward to see how we are using these experiences and the evolution of the Server Side Swift eco-system to improve the operational performance of applications written using the Smoke Framework in areas such as logging, metrics and traceability.

Time and place

Friday, November 1 at 09:00 in Conference room (Salen)

About the speaker

Position

Senior Software Development Engineer at Amazon

Bio

Simon strives to constantly build powerful yet simple architectures, platforms and processes. He is focused on delivering robust and usable developer experiences as a mechanism for enabling simple, elegant and performant applications. Simon is currently a Senior Software Engineer and Engineering Lead at Prime Video, where he is responsible for developing high-scale applications and frameworks.
